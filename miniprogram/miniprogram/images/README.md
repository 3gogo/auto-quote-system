# 小程序图片资源

此目录存放小程序所需的图片资源。

## 需要的图片文件

### TabBar 图标（建议尺寸 81×81px）
- `voice.png` - 报价 Tab 默认图标
- `voice-active.png` - 报价 Tab 选中图标
- `list.png` - 记录 Tab 默认图标
- `list-active.png` - 记录 Tab 选中图标
- `settings.png` - 设置 Tab 默认图标
- `settings-active.png` - 设置 Tab 选中图标

### 页面图标
- `voice-big.png` - 首页欢迎语音图标（建议 160×160px）
- `voice-btn.png` - 录音按钮图标（建议 64×64px，白色）
- `voice-recording.png` - 录音中按钮图标（建议 64×64px，白色）
- `arrow-right.png` - 右箭头图标（建议 32×32px）
- `empty.png` - 空状态图标（建议 200×200px）

## 图标建议

- 使用简洁的线性图标风格
- 默认状态使用灰色 (#999999)
- 选中状态使用主题蓝 (#1677ff)
- 支持透明背景的 PNG 格式

## 推荐图标资源

- [Iconfont](https://www.iconfont.cn/)
- [IconPark](https://iconpark.oceanengine.com/)
- [Heroicons](https://heroicons.com/)
