# 项目上下文 - 小店 AI 报价助手

## 项目概述
- **项目名称**: auto-quote-system（小店 AI 报价助手）
- **目标用户**: 夫妻店/小杂货铺/小型便利店
- **核心目标**: 让男店主在不记价格的前提下独立完成报价与进货核价

## 核心功能
1. **语音报价**: 语音询价 → NLU解析 → 定价引擎 → 语音播报
2. **自动记忆**: 从交易中自动发现商品和顾客，渐进式结构化
3. **定价规则**: 4级规则优先级（专用规则 > 顾客类型 > 商品类别 > 全店默认）
4. **后台复盘**: 女店主轻量校对，每天3-10分钟

## 技术架构
- **后端**: Node.js + Express + TypeScript
- **数据库**: MySQL 8.0 + TypeORM
- **缓存**: Redis（预留）
- **ASR**: Whisper.cpp（本地）
- **TTS**: PaddleSpeech（本地）
- **NLU**: 规则+词典匹配（待实现）

## 数据模型
- `products`: 正式商品表
- `partners`: 顾客/供货商表
- `pricing_rules`: 定价规则表
- `transactions`: 交易记录表
- `candidate_products`: 商品候选表
- `candidate_partners`: 顾客候选表

- 阶段 2 (自动记忆): 正在开发 NLU + 定价引擎
